# LocalStore Notes

> Simple local storage notes.

## Requirements

- Node 0.12.x
- NPM 2.11+
- Angular
- Express 4.x

## Development

Run Gulp to watch and compile Sass

```sh
gulp
```

### Installing Dependencies

Download required dependencies

From within the root directory:
```sh
npm install
```

Start the server

From within the root directory:
```sh
node server/index.js
```
